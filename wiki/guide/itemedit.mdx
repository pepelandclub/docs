Item Edit — плагин и услуга на нашем сервере, позволяющая вам редактировать название и лор любого предмета.

## Допустимые цвета {#allowed-colors}

| 2   | Тёмно-зелёный     | #00AA00 |
| 3   | Тёмно-бирюзовый   | #00AAAA |

## Использование {#how-to-use}

Команда `/ie rename <новое название предмета>` изменяет название предмета

При использовании цветового кода пробел между кодом и названием ставить не требуется

Например, ``/ie rename &4Новое название предмета``

![Пример названия текста](__IMAGESTORAGE__/specimen.png)

Команда `/ie lore add <текст>` добавляет новую строку лора предмета

Строка лора не ограничена по длине, поэтому вам необходимо самостоятельно определить место переноса.
Каждое использование данной команды добавляет одну строку к лору предмета.

![Построчный ввод лора](__IMAGESTORAGE__/lore-row-by-row.png)

Команда `/ie lore set <номер строки> <текст>` изменяет или добавляет строку в лоре предмета с конкретным номером, требуется обязательно указывать конкретный номер строки

Отсчет строк начинается с 1. Можно использовать строки не подряд.

<Callout>
  Не используйте более 10 строк, иначе сервер будет кикать вас
</Callout>

![Лор с отступом](__IMAGESTORAGE__/lore-with-offset.png)

На примере выше были использованы следующие команды:

```
/ie lore add Строка номер 1
/ie lore add Строка номер 2
/ie lore add Строка номер 3
/ie lore set 10 Строка номер 10
```

Команда `/ie lore remove <номер строки>` полностью удаляет строку лора с заданным номером

![Полный лор](__IMAGESTORAGE__/full-lore.png)
После использования команды `/ie lore remove 5`

![Лор после удаления строки](__IMAGESTORAGE__/lore-after-remove.png)

Команда `/ie lore reset` полностью очищает лор предмета
